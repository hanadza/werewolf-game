# werewolf-game

"Village Trial adalah aplikasi game web multiplayer real-time berbasis mekanisme permainan Werewolf. 
Dibangun menggunakan React.js untuk frontend, Node.js + Socket.io untuk backend real-time, dan MySQL sebagai database. Game ini mendukung fitur seperti sistem peran (Werewolf, Seer, Doctor, Villager), voting system, chat real-time, dan win condition detection."
